    Timestamp¶ ↑

    This class converts a String representing a date in RFC2822 format to a Time object and calculate the difference between two such timestamps.

    Usage:¶ ↑

    Timestamp.difference(time1, time2)

    • Receives two timestamps as Strings in RFC2822 format

    • Returns the number of minutes between them.
